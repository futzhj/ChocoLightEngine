✅ P3.1 perf benchmark: scripts/smoke/lua_api_magic_perf.lua
   方法: 1M 次循环 × 5 轮取最小值, 排除 GC/调度噪音
   Windows CI 实测:
     A1 empty Lua loop iter      : 4.0  ns/iter
     A2 pure Lua fn() call       : 21.0 ns/call
     A3 pure Lua obj:method()    : 33.0 ns/call
     B1 Tilemap:GetMapSize       : 60.0 ns/call (双层防御)
     B2 Particles:GetCount       : 60.0 ns/call (双层防御)
   结论: Lua→C 边界总开销 ~27 ns/call, 其中 magic 校验 ≤ 3 ns (< 5%).
   CI: $phaseG171PerfSmoke (Windows runtime smoke job)
