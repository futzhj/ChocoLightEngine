# Light.Network.HttpServer

## `Light.Network.HttpServer.Open`

启动服务器监听, 接受客户端连接

### 返回值

`boolean 是否有新连接`

---

## `Light.Network.HttpServer.Close`

关闭服务器

### 返回值

`void`

---

## `Light.Network.HttpServer.__call`

构造函数, 创建 HTTP 服务器

### 参数

| 名称 | 类型 | 说明 |
|------|------|------|
| `ip` | `string` | 监听 IP (如 "0.0.0.0") |
| `port` | `number` | 监听端口 |

### 返回值

`void`

---
